🎬 Flutter Movie Streaming App

A cross-platform movie streaming app built with Flutter and Dart, powered by Firebase. This app lets users browse, stream, and enjoy movies in real-time, with a modern interface and backend support for authentication, storage, and real-time updates.

🌟 Features

Real-Time Movie Streaming – Stream movies directly from Firebase storage.

User Authentication – Sign up, log in, and manage profiles with Firebase Auth.

Dynamic Movie Catalog – Browse, search, and filter movies in multiple categories.

Favorites & Watchlist – Users can save movies to watch later.

Responsive UI – Works seamlessly on Android and iOS.

Cloud Storage & Database – Uses Firebase Firestore and Storage for dynamic content.

⚙️ Technologies Used

Flutter & Dart – Cross-platform UI framework

Firebase – Authentication, Firestore, Storage, and Hosting

Provider / Bloc – State management

Video Player Plugin – Stream movies inside the app

📄 How It Works

Firebase hosts all movie content and metadata.

Users register and log in through Firebase Authentication.

The app fetches the movie catalog from Firestore in real-time.

Users can stream movies, add to favorites, or browse categories.

All user activity is synced with Firebase for persistence.
